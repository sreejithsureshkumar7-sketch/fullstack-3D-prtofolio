# 3D Full Stack House Portfolio

A unique portfolio website where the house front means frontend and the basement/foundation means backend.

## Run locally
```bash
npm install
npm start
```
Open: http://localhost:5000

## Optional MongoDB
Create `.env` file from `.env.example` and add your MongoDB URI.

## Deploy
- Render: use this full Node project.
- GitHub Pages: upload only `public` folder files for frontend-only version.
